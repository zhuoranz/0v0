1.0 基础数据类型

1.1 byte
位数：8位、有符号的，以二进制补码表示的整数
（2进制补码？）
最大值： 127（2^7-1）
最小值：-128（-2^7）
默认值：    0


  一个数如果是正，则它的反码与原码相同；
        一个数如果是负，则符号位为1，其余各位是对原码取反；
1.2 int
位数：32位、有符号的,以二进制补码表示的整数
（2进制补码？）
最大值： 2,147,483,647（2^31 - 1） 
最小值：-2,147,483,648（-2^31）
默认值：    0
1.3 char
位数：一个单一的16位Unicode字符；
最小值： ’\u0000’（即为0）
最大值： ’\uffff’（即为65,535）
char数据类型可以储存任何字符；
例子：char letter = ‘A’。

2.0 基础运算

2.1 取余运算符 %
  • 整数
    a%b=a-(a/b)*b
    因此，负数 ％ 正数，相当于是负数绝对值％正数加符号
  • 浮点数
    a%b=a-(b*q)， q=int(a/b)
    因此，相当于是证书的取余＋小数部分
2.2 bitwise operator

a = 0011 1100
b = 0000 1101

0xff 16进制15*16＋15 ＝ 255 ＝ 1111 1111
