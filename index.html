葱姜炒龙虾

食材

  • 龙虾
  • 葱
  • 姜
  • 料酒（花雕，应该都可以）
  • 白胡椒粉
  • 盐（非常少）
  • 高汤水（可选）
  • 油
  • 淀粉

Step 0 准备龙虾

把龙虾斩件，按照大概关节切，去掉排泄物

Step 1 裹淀粉炸制

斩件后，厨房用水洗点水分，裹淀粉，有的可以压一下壳，粘更多淀粉。油热炸制肉白开始发黄，称出备用

Step 2 炒

保留少许底油，放入姜煸香后，放入葱段。炒香后倒入炸好的龙虾，沿锅边一周倒入料酒，加入白胡椒粉和少量高汤。加盖炖2分钟左右，大火收汁，放入少许盐，也可不放。翻滚后即可出锅

扁豆焖面

食材

  • 糖（可选）
  • 肉 （我选择的是有肥肉的后臀尖，但是都可以，有肥肉会香一些）
  • 八角
  • 豆角
  • 葱
  • 老抽
  • 盐
  • 花椒
  • 水
  • 油
  • 蒜
  • 米醋

Step 0 (可选) 准备花椒水

花椒水： 放花椒然后用热水或者温热的水泡开，泡5－10分钟

Step 1 (可选) 炒糖色

炒糖色实际的意义是在与上色。所以，单纯的使用老抽也是可以达到目的的。
炒糖色可以选择用油炒，水炒或者水油炒。重点在于小火，用油炒注意水少可
能会溅出来。反正，等到糖变为焦糖色即可。

Step 2 炒肉

炒好的糖色里面炒肉，让肉上上颜色，加入八角（别名大料），炒香，加入葱、老抽、盐。

如果没有炒糖色，可以直接，倒油放八角、葱炒香，后炒肉，如果有肥肉，可以先炒肥肉，让他出油，然后再放瘦肉。

Step 3 炒豆角煮汤

把豆角放入锅中，炒到豆角已经快熟了的时候，加入花椒水（留一些后来做调味汁）和一般的水煮汤，煮出肉香味。加入油和盐。煮出香味之后，改小火。把汤倒出来一些，只留少量的汤在锅底。

其实也可以，不倒出来，只是多次加更容易入味。

Step 4 加入面

将菜铺平，放上面，汤快干的时候，加入烧好的汤，多次焖面。

Step 5 加入调味汁

最后快出锅的时候，淋上花椒水、蒜末、米醋的混合调味汁，闷3分钟，拌一下，就可以吃啦～

使用GitHub和GitHub Pages搭建博客

0. GitHub Pages 和 Hexo

GitHub Pages是GitHub推出的一款免费的静态网站发布服务。网站的文件部署在GitHub repository (直译是仓库，实质上是一个储存空间) 上，GitHub用户可以通过GitHub Pages建立个人或项目的网页。如果你对英文比较了解，GitHub官网上的教程简单明了，可以点击查看。

GitHub支持使用纯HTML来编辑页面。同时，也支持静态网站生成器，也可以说是一种架构来帮助你建静态网站。Jekyll和Hexo是两种非常流行的工具。通过他们，用户可以简单地使用Markdown发布博客。而且，这两种工具都有开源的主题可以选择。

GitHub Pages 官方教程使用Jekyll作为静态网站生成器。但是，我使用的是Hexo作为我的静态网页生成工具。
